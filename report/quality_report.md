# Отчёт о качестве данных

## 1. Введение

**Датасет:** Данные о продажах товаров  
**Источник:** Файл `ПРОДАЖА_data (1).xlsx`  
**Размер:** 500 строк × 9 столбцов  

**Предметная область:** Розничная торговля — записи о продажах электроники и аксессуаров (наушники, ноутбуки, телефоны, зарядки, чехлы)  

**Условный заказчик:** Отдел аналитики интернет-магазина  

**Возможные задачи ИАД:**
1. Прогнозирование выручки по дням/месяцам
2. Кластеризация покупателей по возрасту и способу оплаты
3. Анализ сезонности продаж по городам

## 2. Паспорт датасета

| Столбец | Тип | Описание | Примечания |
|---------|-----|----------|------------|
| id | int64 | Номер транзакции | Уникальный, без пропусков |
| дата | datetime | Дата покупки | Диапазон: 01.01.2024 — 14.05.2025 |
| товар | object | Наименование товара | 5 категорий: Наушники, Ноутбук, Телефон, Зарядка, Чехол |
| город | object | Город покупки | 5 городов: Москва, СПб, Казань, Екатеринбург, Новосибирск |
| количество | int64 | Количество единиц | От 1 до 100 (есть выброс — 100 шт.) |
| цена | int64 | Цена за единицу в рублях | Цены рыночные (от 500 до 100 000 руб.) |
| скидка_% | float64 | Скидка в процентах | Много пропусков (≈15% строк) |
| возраст | float64 | Возраст покупателя | Много пропусков (≈40%), есть выброс 150 лет |
| оплата | object | Способ оплаты | 3 варианта: онлайн, карта, наличные |

## 3. Основные проблемы качества данных

| № | Тип проблемы | Описание | Потенциальный риск | Предлагаемое решение |
|---|--------------|----------|-------------------|---------------------|
| 1 | Пропуски | `возраст`: ~40% пропусков | Невозможно анализировать возрастные группы | Заполнить медианой (35 лет) |
| 2 | Пропуски | `скидка_%`: ~15% пропусков | Искажение расчёта итоговой цены | Заполнить 0% (без скидки) |
| 3 | Выбросы | `возраст` = 150 лет (строка 151) | Нереалистичное значение | Заменить на медиану (35 лет) |
| 4 | Выбросы | `количество` = 100 шт. (строка 101) | Сомнительная крупная покупка | Проверить, возможно ошибка ввода |
| 5 | Аномалии | `город` = "СПб" вместо "Санкт-Петербург" | Неединообразное наименование | Привести к полному названию |
| 6 | Дубликаты | Нет полных дубликатов строк | — | — |

## 4. План улучшения качества

**Приоритет 1 (критично):** Исправить возраст 150 лет (заменить на медиану)

**Приоритет 2:** Заполнить пропуски в возрасте (медианой) и скидке (0%)

**Приоритет 3:** Проверить выброс "количество = 100", нормализовать названия городов (СПб → Санкт-Петербург)

## 5. Краткое резюме

**Что узнали о данных:**  
Датасет содержит 500 записей о продажах за ~16 месяцев. Цены в датасете приведены к рыночным значениям (ноутбуки: 50 000-100 000 руб., чехлы: 500-2 000 руб.). Данные в целом пригодны для анализа, но требуют предобработки.

**Основные проблемы:**  
- **Пропуски** в возрасте (~40%) и скидке (~15%)
- **Выбросы:** возраст 150 лет, количество 100 штук

**Особенности, требующие внимания:**  
- Популярные товары: равномерно распределены между категориями
- Популярные города: Москва и Санкт-Петербург
- Способы оплаты: онлайн, карта и наличные примерно поровну
