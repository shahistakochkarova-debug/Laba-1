# Лабораторная работа №1: Анализ качества данных

## Описание проекта
Первичный анализ датасета "Продажи товаров интернет-магазина" в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:** розничная торговля электроникой и аксессуарами (наушники, ноутбуки, телефоны, зарядки, чехлы). Датасет содержит 500 транзакций за 2024-2025 гг. с информацией о товарах, городах, ценах, скидках, возрасте покупателей и способах оплаты. Возможные задачи ИАД: прогнозирование объёма продаж, кластеризация покупателей по возрасту, анализ сезонности и региональных предпочтений.

**Цель лабораторной:** аудит качества данных, построение паспорта датасета, выявление проблем и предложение способов их исправления.

## Структура проекта
- `data/` — файл с данными (ПРОДАЖА_data.xlsx)
- `notebooks/` — Jupyter Notebook с анализом (`1_data_understanding.ipynb`)
- `report/` — текстовый отчёт о качестве данных (`quality_report.md`)

## Выявленные проблемы
1. Пропуски в столбце "возраст" (~40%)
2. Пропуски в столбце "скидка_%" (~15%)
3. Нереалистичный возраст 150 лет (строка 151)
4. Выброс в количестве (100 штук, строка 101)
