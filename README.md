# Лабораторная работа №1: Анализ качества данных

## Описание проекта
Анализ данных о продажах товаров (наушники, ноутбуки, телефоны, зарядки, чехлы).  
**Цель:** проверить качество данных, найти ошибки и предложить исправления.

## Структура проекта
- `data/` — файл с данными (ПРОДАЖА_data.xlsx)
- `notebooks/` — Jupyter Notebook с анализом
- `report/` — текстовый отчёт о качестве данных

## Выявленные проблемы
1. Пропуски в столбце "возраст" (~40%)
2. Пропуски в столбце "скидка_%" (~15%)
3. Отрицательная цена (строка 201)
4. Нереалистичный возраст 150 лет (строка 151)
